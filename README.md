# LLM Chatbot Frontend

A modern web interface for interacting with a Large Language Model (LLM) powered chatbot.
This frontend provides a responsive and user-friendly chat interface that communicates with the backend API to generate intelligent responses using LLM and Retrieval-Augmented Generation (RAG).

---

## Features

* Interactive chatbot interface
* Real-time response display from LLM backend
* Adjustable parameters for model responses
* Clean and responsive UI
* Modular React component structure
* Easy integration with AI/ML backend services

## Installation and Setup

### 1. Clone the Repository

```
git clone https://github.com/your-username/LLM-Frontend.git
```

### 2. Navigate to the Project Folder

```
cd frontend
```

### 3. Install Dependencies

```
npm install
```

### 4. Start Development Server

```
npm run dev
```

The application will run at:

```
http://localhost:5173
```

---

## Backend Integration

This frontend connects to a backend API that handles:

* LLM inference
* Retrieval-Augmented Generation (RAG)
* Knowledge base queries
* Response generation

Make sure the backend server is running before starting the frontend.
